# 創建你的第一個存在證明

歡迎來到 Proof of Existence！本指南將帶你創建你的第一條光軌——一個永久記錄在區塊鏈上的存在證明。

## 你需要準備

- 網頁瀏覽器（Chrome、Firefox、Safari 或 Edge）
- 就這樣！不需要任何加密貨幣知識。

## 第一步：登入 / 連結錢包

訪問 [proofexistence.com](/) 並點擊右上角的「登入 / 連結錢包」。你有兩個選擇：

### 社交登入（推薦新手使用）
使用你現有的帳號登入：
- Google
- Facebook
- Apple
- Line
- X（Twitter）

**不需要加密錢包！** 當你使用社交登入時，Web3Auth 會自動為你創建一個安全的區塊鏈錢包。你無需任何技術設置就能成為 Web3 用戶。

### 外部錢包
如果你已經有加密錢包，可以直接連結：
- MetaMask
- WalletConnect
- 其他 Web3 錢包

## 第二步：進入畫布

點擊 [Draw to Proof](/canvas) 進入 3D 畫布。你會看到一個黑暗的空間——這是你的創作天地。

## 第三步：繪製你的光軌

- 在畫布任意位置**點擊並按住**開始繪製
- **移動滑鼠**同時按住來創建軌跡
- **放開**暫停繪製
- 繪製至少 **10 秒**才能創建有效證明

你的軌跡是獨一無二的藝術表達，將被永久記錄。花點時間，創造一些有意義的東西！

## 第四步：提交你的證明

繪製超過 10 秒後，點擊「Submit Proof」選擇證明類型：

### 標準證明（免費）

**適合：** 想要免費參與的所有人。

| 特點 | 詳情 |
|------|------|
| **費用** | 完全免費 |
| **驗證時間** | 每日 UTC 00:00 批次處理 |
| **技術** | Merkle 樹聚合 |
| **上鏈** | 是，透過每日批次結算 |
| **存儲** | 協議管理 |

**運作原理：** 你的證明會與當天創建的其他證明一起被整合到 Merkle 樹中。根哈希在 UTC 午夜提交到 Polygon 區塊鏈，以密碼學方式證明你的軌跡在該時間存在。這種高效的批次處理意味著你不需要付任何費用——協議會承擔 Gas 費。

### 永恆證明（付費）

**適合：** 想要即時驗證並完整擁有 NFT 的人。

| 特點 | 詳情 |
|------|------|
| **費用** | POL 或 TIME26 代幣 |
| **驗證時間** | 即時（幾秒內） |
| **NFT** | 是，ERC-721 代幣鑄造到你的錢包 |
| **存儲** | 永久存儲在 Arweave（去中心化） |
| **徽章** | 專屬收藏家徽章 |

**運作原理：** 你的軌跡會立即上傳到 Arweave（永久、去中心化存儲）並在 Polygon 上鑄造為 NFT。你將獲得證明的完整所有權，作為可交易的數位資產，還有專屬的個人檔案徽章。

**定價：** 基本費用 + 超過 45 秒的每秒費用。前 45 秒包含在基本費用中。

**支付方式：**
- **POL** - Polygon 原生代幣
- **TIME26** - 我們的項目代幣（透過推薦和互動賺取）
- **無 Gas 鑄造** - 使用未領取的 TIME26 餘額，無需錢包簽名

## 下一步

提交後，你可以：
- 在[探索](/explore)頁面查看你的證明
- 分享到社交媒體（X、Threads、Instagram）
- 在 [Cosmos](/cosmos) 3D 視覺化中找到它
- 下載截圖保存或分享

恭喜！你剛剛在區塊鏈上留下了永久的印記。歡迎加入 2026 集體藝術實驗。
