# 存在の証明：現代の叙事詩

## 1. 哲学：なぜ私たちはここにいるのか

### 忘れ去られる者と永遠なる者

歴史の記録において、不滅性はごく少数の人々にのみ許された特権でした。王は石に名を刻み、征服者は叙事詩に自らを記し、詩人はインクで永遠のものとなりました。

しかし、残りの私たち——生き、愛し、苦闘した何十億もの人々——にとって、歴史は沈黙しています。私たちは亡霊なのです。

デジタル時代において、この儚さは加速しています。サーバーがクラッシュし、プラットフォームが閉鎖され、サブスクリプションが期限切れになると、私たちの デジタルな足跡は虚無へと消え去ります。私たちはオンラインで何千時間も過ごし、創作し、交流し、生きていますが、電源が切られれば、すべてはゼロに戻ります。

**Proof of Existence (POE)** は、この沈黙に対する反逆です。
それは「デジタルノマド」から「デジタル永続性」への移行です。あなたの存在、あなたの時間、そしてあなたのエネルギーが、儚い亡霊ではなく、永遠に刻まれた歴史であることを保証する暗号化メカニズムなのです。

### 現代の叙事詩

私たちは**現代の叙事詩**を構築しています。
一人の英雄のために一人が書いた昔の叙事詩とは異なり、これは何百万人もの人々によって書かれる集合的な傑作です。

- **あなたは英雄です：** あなたのすべての動きが一つの詩（うた）となります。
- **ブロックチェーンは石碑です：** 消去することのできない不変の台帳です。
- **アーカイブは永遠です：** アーウィーブ（Arweave）というパーマウェブに保存され、数世紀後のデジタル考古学者によって目撃されることになります。

## 2. メカニズム：どのように機能するか

これは単なるアートプロジェクトではありません。「生存証明（Proof of Life）」を取り込むためのプロトコルです。

### ステップ 1：接続

ソウルバウンドウォレット（Soulbound Wallet）を接続します。これはあなたのデジタルIDであり、このメタバースにおけるあなただけの識別子です。

### ステップ 2：虚無への入場

無限で暗い3D空間——私たちの「グローバルキャンバス」に入ります。ここでは、時間が唯一の指標です。

### ステップ 3：刻印

カーソルを動かすとき、あなたはただ線を引いているだけではありません。

- システムは、あなたの位置ベクトル `(x, y, z)` とタイムスタンプ `(t)` をミリ秒単位の精度で取得します。
- すべての躊躇、すべての加速、すべての円は、その瞬間のあなたの潜在意識の状態を反映しています。
- これらのデータポイントは、ユニークな「光の軌跡（Light Trail）」へと変換されます。

### ステップ 4：ミントと永遠化

一般の方々とコレクターの方々の両方に対応するため、「デュアルパス」ミントメカニズムを提供しています。

**1. 標準証明（Standard Proof） —— [無料]**
一般の参加者に最適です。

- **メカニズム：** データは一時的に私たちのサーバーに保存され、UTC 00:00に毎日オンチェーンに提出するためにマークルツリーにバッチ処理されます。
  - **ストレージ：** プロトコルによってデータバッチ全体が **Arweave/IPFS** にアップロードされ、分散型のバックアップが保証されます。
- **報酬：** 「時間の証明（Proof of Time）」として **Time26** トークンを獲得します。
- **コスト：** **ガス代 0**。すべてのガス代とストレージ費用は私たちが負担します。

**2. 永続証明（Perpetual Proof） —— [有料]**
クリプトネイティブやコレクターに最適です。

- **メカニズム：** あなたの軌跡は即座に **Arweave** に書き込まれ、**Polygon** スマートコントラクトによってリアルタイムで検証されます。
- **メリット：**
  - 個人の軌跡 **NFT** を即座にミントできます。
  - 「インスタントミンター（Instant Minter）」バッジを獲得できます。
  - 公式サーバーから独立した、完全に分散化されたデータ所有権が得られます。
- **コスト：** 少額の手数料が必要です（下記の「存在するための支払い」を参照）。

## 3. 技術アーキテクチャ

「時代を超えて続く」という約束を果たすために、私たちは最も堅牢な分散型技術スタックを選択しました。

| レイヤー             | 技術                      | 役割                                                                                          |
| :------------------- | :------------------------ | :-------------------------------------------------------------------------------------------- |
| **インターフェース** | **Next.js 16 + Three.js** | WebGLを使用した没入型の3Dインタラクションを提供し、高性能なレンダリングを実現します。         |
| **コンセンサス**     | **Polygon (PoS)**         | 所有権の最終的な仲裁者として機能し、迅速な確認と低コストを保証します。                        |
| **ストレージ**       | **Arweave**               | 「パーマウェブ」。一度アップロードされたデータは永久に保存され、理論上は200年以上持続します。 |
| **アイデンティティ** | **Clerk**                 | 従来のWebと分散型の世界をつなぐ、シームレスなWeb3ログイン体験を提供します。                   |

**なぜArweaveなのか？**
従来のクラウドストレージ（AWS, Google Cloud）は「賃貸型」です。支払いをやめればデータは削除されます。Arweaveは「寄付型」です。一度支払えば、永久に保存されます。デジタルの遺産を築くための唯一の選択肢です。

## 4. 経済圏：通貨としての時間（トークノミクス）

### Time Token (Time26)

**Time26** はエコシステムのネイティブトークンであり、コミュニティによって貢献された「アテンション・タイム（注目時間）」を表しています。

- **シンボル:** $TIME26
- **総供給量:** 48,420,000
  - _コンセプト:_ 2026年は31,536,000秒あります。インセプション（開始）期間中にコミュニティとチームが費やした集合的なエネルギー（約1.54年分の時間）を考慮して、この「1年」を拡張しました。

### なぜ 48,420,000 なのか？（時間の数学）

この数字は、「基準年」と「エコシステム拡張分」で構成されています。

**1. 基準年**

- **31,536,000 (65%)** = **ユーザー報酬**
  - 2026年のすべての秒数（365日）と一致します。これは純粋にクリエイターのためのものです。

**2. エコシステム拡張分**
プロジェクトのインフラを支えるために追加された「余分な時間」（合計 16,884,000）：

- **10,000,000 (21%)** = **流動性プール**
  - Uniswapや市場のために、Time26が取引可能であることを保証します。
- **4,730,400 (10%)** = **チーム**
  - 構築者へのインセンティブと長期的なメンテナンスのため。
- **2,153,600 (4%)** = **運営準備金**
  - サーバー費用、イベント報酬、緊急資金のため。

**合計:** 31.5M (ユーザー) + 16.9M (エコシステム) = **48,420,000**

### Time26の獲得方法は？（マイニング）

**「存在マイニング（Existence Mining）」**を通じて無料でトークンを獲得できます。
獲得量は、あなたが投資した時間と、あなたがその時いるセクターの混雑状況によって決まります。

**希少性の公式:**
`ユーザー報酬 = Σ (1 Time26 / その秒、そのセクターにいる人数)`

- 混雑した「タイムズスクエア」のようなエリアで描くと、Time26はごくわずかしか得られません。
- 誰もいない荒野で描けば、その時間の全価値を得ることができます。
- これにより、キャンバス全体に光の軌跡が均等に分布するように動機づけられます。

### 永遠のコスト

### サステナビリティ・バルブ（持続可能性の弁）

私たちは従来の暗号資産の「ポンプ・アンド・ダンプ（価格のつり上げと急落）」のサイクルを拒絶します。私たちはプロトコルの中心に**ダイナミック・バルブ**を組み込みました。

1.  **デフレモード（フェーズ A）：**
    - _いつ:_ 初期段階または供給が安定している時。
    - _メカニズム:_ プロトコル収益の100%（例：上記の$2〜$20の手数料）を使用して、公開市場から$TIME26を買い戻し、**バーン（焼却）**します。
    - _結果:_ 希少性が高まります。あなたが「保有している時間」の価値が上がります。

2.  **サステナビリティモード（フェーズ B）：**
    - _いつ:_ 供給が極端に低下（3000万未満）するか、価格が持続不可能なほど急騰した時。
    - _メカニズム:_ 20% バーン / 80% トレアジャリー（財務）。
    - _結果:_ トレアジャリーは**2027年のグローバル展覧会**に備えるための十分な資金を獲得し、プロジェクトがハイプサイクルを超えて存続することを保証します。
